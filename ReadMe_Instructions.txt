About : React App is about tv show Vikings, you will find 6 blog posts related to vikings tv show, and on navigation side, you will find
latest news headlines from google news.

How to run this app :
1. Install Node js and npm through nodejs website.
2. Open files in code editor (recommended Visual Code)
3. Install react-router-dom through terminal if not already installed.
4. Go to Google Api, and get the api key.
5. Open nav.js, and put the api key where its mentioned.
6. Open App.js which is main app.
7. Go to terminal or press ctr+shift+` in windows to open terminal in visual code.
8. type "npm start"
9. React App will run on browser.
